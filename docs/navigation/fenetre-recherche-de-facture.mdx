Quand vous voulez retrouver vos factures de revenus et/ou de dépenses,
tout ce trouve dans la fenêtre de recherche de factures (aussi connue sous le nom "Mes factures").

## Ouvrir la fenêtre de recherche de facture pour voir vos factures existantes

:::note
Quand vous êtes à la fenêtre principale du logiciel,
le raccourci clavier pour ouvrir la fenêtre
de recherche de factures existantes est&nbsp;:&nbsp;&nbsp;<kbd>Ctrl</kbd>+<kbd>F</kbd>.
:::
