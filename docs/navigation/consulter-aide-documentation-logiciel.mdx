Pendant que vous travaillez dans votre tenue de livres et que
vous avez une question à propos du logiciel, cliquez sur le bouton d'aide et documentation
du menu supérieur afin de consulter rapidement la documentation du logiciel.

Si vous ne trouvez pas réponse à votre question,
écrivez-nous à admin@finance-d.com en précisant les détails de votre interrogation
pour que nous puissions vous aider le mieux possible.
