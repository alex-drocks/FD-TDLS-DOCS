Pour ouvrir un dossier existant, sélectionnez le nom du dossier
que vous voulez dans votre liste. Ensuite, entrez le mot de passe
et cliquez sur le bouton "Ouvrir le dossier".

<Image img={screenshot}/>

## 1. Sélectionnez le nom du dossier à ouvrir
Dans la première case (celle avec l'icône de dossier), sélectionnez le dossier que vous voulez ouvrir.

## 2. Entrez le mot de passe du dossier
Entrez le mot de passe du dossier.
Vous avez déterminé ce mot de passe au moment de la création de ce dossier.

## 3. Cliquez sur "Ouvrir le dossier"
Une fois que vous avez choisi le dossier que vous voulez ouvrir et entré son mot de passe,
il suffit de cliquer sur le bouton "Ouvrir le dossier".

Si un message d'erreur vous indique que ce n'est pas le bon mot de passe.
Réessayez. Si vous l'avez oublié, utilisez le bouton de récupération à droite.

## Terminé
Quand tout c'est bien passé, le logiciel va ouvrir le dossier et vous serez prêts à y travailler.

Pour obtenir plus d'informations à propos des dossiers de tenue de livres dans Finance&nbsp;D, consultez la
section <Link to="/docs/gestion-dossiers/intro-gestion-dossiers">Gestion des dossiers de tenue de livres</Link>.
