Les dossiers dans Finance D représentent un classeur où toutes vos factures sont enregistrées sécuritairement.

Vous pouvez créer un ou plusieurs dossiers selon vos besoins. Chaque dossier est indépendant et sécurisé
par son propre mot de passe. Vous pourriez par exemple gérer deux entreprises différentes en leur créant
chacun un dossier.

Pour créer un nouveau dossier, choisissez un nom (par exemple, le nom de votre entreprise),
un mot de passe pour protéger vos données, et votre adresse courriel pour récupérer le mot de passe si
jamais vous l'oubliez. Ensuite, cliquez sur le bouton "Créer le dossier".

<Image img={create}/>


## 1. Choisissez un nom de dossier
Le nom du dossier Finance D sert à l'identifier. Ce nom doit être unique.
Vous ne pouvez donc par avoir deux dossiers du même nom.
La prochaine fois que vous ouvrirez le logiciel, vous verrez ce nom
dans la liste de vos dossiers afin de l'ouvrir.

## 2. Choisissez un mot de passe
Le mot de passe du dossier Finance D sert à protéger vos informations en les encryptant.
De cette façon, une personne qui ne connait pas votre mot de passe ne pourra pas voir vos données.

## 3. Choisissez un email de récupération
Puisque chaque dossier Finance D est sécurisé par mot de passe,
il pourrait arriver que vous l'oubliez.
Ce courriel de récupération servira à vous envoyer un code de
validation pour changer votre mot de passe.
N'utilisez pas un faux courriel car vous ne pourrez pas recevoir le moyen de changer de mot de passe.

Notez bien que le logiciel Finance D ne divulge aucune information.
Nous-mêmes ne savons pas quelles informations vous utilisez.


## 4. Cliquez sur "Créer le dossier"
Lorsque vous avez choisi le nom, le mot de passe ainsi que l'email de récupération du dossier Finance D,
cliquez sur le bouton "Créer le dossier".

## Terminé
Si tout c'est bien passé, le logiciel affichera un message de succès pour la création de votre nouveau dossier.
Il est maintenant possible d'ouvrir ce dossier et de commander à travailler dans Finance D.
