Vous pouvez supprimer vos factures en les ouvrant à partir de la fenêtre de recherche
et en cliquant sur l'option "Supprimer" dans le menu du coin inférieur droit de la fenêtre principale.

## 1. Chargez la facture à supprimer
Chargez d'abord la facture que vous voulez supprimer dans la fenêtre principale.
Si vous n'êtes pas certain de comment vous y prendre, consulter le document
[Retrouver et modifier vos factures](trouver-vos-factures.mdx).


## 2. Cliquez sur le menu d'options
Quand la facture est affichée dans la fenêtre principale, une option sera disponible
dans le menu du coin inférieur droit. Cliquez sur le bouton du menu pour l'ouvrir.
<Image img={optionsMenuBtn}/>
<br/>


## 3. Cliquez sur l'option "Supprimer"
Cliquez sur "Supprimer" dans le menu d'options et confirmez que vous voulez bel et bien
l'effacer lorsqu'il vous sera demandé de le faire.
<Image img={deleteInvoice}/>
<br/>


<br/>
<br/>
<hr/>

## Terminé
Quand vous avez suivi les étapes ci-dessus et supprimé une facture avec succès,
la facture n'existera plus et aucune trace n'y fera référence dans le logiciel.
Les montants que la facture contenaient ne seront donc plus comptabilisés.
En date du 2021-04-19, supprimer une facture est irréversible.
