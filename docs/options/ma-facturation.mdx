Les options dans l'onglet "Ma facturation", concernent votre facturation client.
C'est dans cet onglet de la fenêtre des options que vous pouvez définir vos préférences
par rapport à la gestion de vos factures de revenus.

## Quelles sont vos préférences concernant pour vos nouveaux clients ?
Vous pouvez définir quel code taxe et quelle catégorie financière que vous voulez utiliser par défaut.
Donc, quand vous allez créer un nouveau client, Finance&nbsp;D utilisera toujours votre choix.

## Comment Finance&nbsp;D doit-il générer vos numéros de factures ?
Si vous voulez personnaliser la manière dont le logiciel génère votre séquence de numéros de factures de revenus,
utilisez ces préférences. Il est possible d'utiliser un préfixe ainsi que l'année en cours et une longueur minimale.
