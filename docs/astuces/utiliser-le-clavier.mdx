Le meilleur moyen d'aller plus vite dans Finance&nbsp;D, est d'utiliser votre clavier au maximum.

L'interface de création de factures, est l'endroit idéal pour mettre ce truc en pratique.
Vous allez gagner du temps en créant vos factures à l'aide du clavier.
En fait, quand on devient un peu plus habitués,
la souris n'est même plus nécessaire pour créer des factures.


## Création rapide de factures avec le clavier

### Le bouton intelligent <kbd>Enter ↵</kbd>
Comme nous voyons dans l'exemple ci-haut,
le plus rapide, est d'appuyer sur <kbd>Enter</kbd>
dès que vous voulez passer à <strong>la prochaine étape logique</strong>.

En fait, le bouton <kbd>Enter</kbd> confirme l'action en cours et valide le statut de la facture
pour vous diriger automatiquement à l'endroit où vous êtes rendus.

Par exemple : Quand vous écrivez un nom de client et appuyez sur <kbd>Enter</kbd>, vous allez être envoyé
à la date de facture parce que votre numéro de facture pour les revenus est déjà correcte.

À l'inverse, si vous avez appuyé sur <kbd>Enter</kbd> par erreur,
utilisez <kbd>Shift</kbd>+<kbd>Enter</kbd> pour retourner à l'étape précédente.

  relativePath="astuces/astuces-shift-enter-go-back-step.gif"

### Le bouton <kbd>Tab ↹</kbd> pour avancer et reculer
En utilisant le bouton <kbd>Tab</kbd>, vous allez passer à la prochaine case disponible sans skipper
les cases qui sont déjà valides (comme c'est le cas avec le bouton <kbd>Enter</kbd>).

Servez-vous de <kbd>Tab</kbd> pour avancer d'une étape,
ou de <kbd>Shift</kbd>+<kbd>Tab</kbd>
pour retourner en arrière d'une case.

### Les flèches <kbd>&larr;</kbd> <kbd>&rarr;</kbd> <kbd>&uarr;</kbd> <kbd>&darr;</kbd> pour sélectionner
Un bon moyen de précsier votre choix dans une liste déroulante où il reste plus d'une option,
c'est d'utiliser vos flèches pour aller de haut en bas.

Pour les calendriers où l'on doit choisir une date, les flèches fonctionnent bien aussi.
Mais, parfois, c'est plus rapide d'écrire la date au clavier.

### Le bouton <kbd>Esc</kbd> pour fermer un menu popup ou une liste déroulante
Quand une fenêtre de type "popup" est ouverte par-dessus l'interface principale,
appuyez sur <kbd>Escape</kbd> pour canceler / fermer le menu instantanément.

<LazyGif
  relativePath="astuces/astuces-escape-cancel.gif"
  alt="Démo de l'utilisation du bouton 'Escape' pour fermer un popup ou canceler une action"

### Le bouton <kbd>+</kbd> pour créer une nouvelle ligne de transaction
Quand votre curseur est dans une ligne de transaction d'une facture,
utilisez le bouton <kbd>+</kbd> pour ajouter une nouvelle ligne sans avoir besoin
de cliquer avec la souris.


### Le bouton <kbd>*</kbd> pour aller à la colonne QTÉ
Quand votre curseur est dans une ligne de transaction d'une facture,
utilisez le bouton <kbd>*</kbd> pour aller directement à la colonne de quantité
de cette ligne. Cela va vous permettre d'utiliser votre pavé numérique
comme une calculatrice.

Exemple : 100 * 2 + 50 = ligne de transaction avec le montant 100$ fois 2
avec une nouvelle ligne de transaction au montant de 50$.

Bref, quand vous êtes dans la fenêtre principale où vous gérez vos factures de revenus et de dépenses,
servez-vous autant que possible de votre clavier pour naviguer dans l'interface du logiciel le plus rapidement possible.
Vous allez même remarquer qu'à la fin nous sommes moins fatigués qu'en utilisant la souris pour chaque action.
