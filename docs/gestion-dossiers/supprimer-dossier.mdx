Quand vous n'avez plus besoin d'un dossier de tenue de livres,
il est possible de le supprimer du logiciel Finance D complètement.
Ainsi, il ne sera plus visible dans votre liste des dossiers de la fenêtre de connexion.

Voici comment supprimer un dossier&nbsp;:

1. [Ouvrez d'abord le dossier](/docs/bases/ouvrir-dossier) que vous voulez supprimer.
2. Ensuite, [ouvrez la fenêtre d'options](/docs/navigation/fenetre-options-preferences-parametres).
3. Cliquez sur l'onglet "Mon dossier".
4. Dans la section "Zone danger", cliquez sur le bouton "Supprimer ce dossier complètement".
5. Écrivez ou copiez-collez le nom exact du dossier.
6. Finalement, cliquez sur le bouton "Supprimer" pour confirmer la suppression.

Si le dossier est bien supprimé, le logiciel va redémarrer et ce dossier n'existera plus du tout.
Il ne devrait donc plus être visible dans la liste de dossiers de votre fenêtre d'accueil / fenêtre de connexion.
