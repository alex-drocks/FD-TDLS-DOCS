Si vous voulez gagner du temps et que l'aspect sécuritaire est moins important pour vous,
vous pouvez à tout moment décider de mémoriser le mot de passe de votre dossier de tenue de livres
pour ne pas avoir besoin de le récrire à chaque connexion.

## Voici comment mémoriser le mot de passe de votre dossier dans Finance&nbsp;D&nbsp;:

1. Ouvrez le logiciel à la page d'accueil / connexion.
2. Sélectionnez votre dossier de tenue de livres.
3. Cliquez sur l'option "Mémoriser mon mot de passe pour la prochaine fois".
4. Connectez-vous comme d'habitude à votre dossier.
5. Terminé. À votre prochaine connexion, le mot de passe sera déjà écrit.

:::note
L'option de mémorisation de mot de passe est unique pour chaque dossier de tenue de livres.
Donc, si vous êtes plus d'une personne à utiliser un même ordinateur, une personne peut choisir
de mémoriser son mot de passe et l'autre peut choisir de ne pas le mémoriser.
:::

## Risque à considérer en utilisant la mémorisation de mot de passe
Quand vous choisissez de mémoriser votre mot de passe pour la prochaine connexion,
toute personne ayant physiquement accès à votre ordinateur pourrait se connecter puisque le mot de passe est prérempli.
Si cela ne vous importe pas, il n'y aucun risque. Vos enregistrements restent encryptés
et sécures pour le monde extérieur et pour les partages de copies de sauvegardes.

## Désactiver la mémorisation de votre mot passe
Pour désactiver la mémorisation de votre mot de passe,
il suffit de désélectionner l'option
"Mémoriser mon mot de passe pour la prochaine fois".
En faisant cela, la prochaine fois que vous ouvrirez le logiciel,
le mot de passe devra être écrit manuellement.
