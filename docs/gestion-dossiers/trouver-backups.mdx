L'endroit où vos copies de sauvegardes sont enregistrées dans votre ordinateur, est géré par le logiciel automatiquement.
S'il arrive que vous ayez besoin de connaitre cet emplacement, suivez les instructions ci-dessous.

## Voici comment trouver vos backups de Finance&nbsp;D&nbsp;:

1. Dans n'importe quelle fenêtre du logiciel, appuyez sur le raccourci clavier <kbd>Alt</kbd>+<kbd>B</kbd>.
2. Le dossier de votre ordinateur qui contient vos backups devrait s'ouvrir.
3. Ouvrez l'un des sous-dossiers qui est un identifiant unique du genre "b7310bdb-50f0-41cd-9b7d-a777d6f2cc22".
4. Si vous avez des copies de sauvegardes, vous verrez des **fichiers .zip** à l'intérieur.
5. **Faites une copie du fichier .zip** le plus récent, pour le déplacer dans un autre dossier.

### Mise en garde
:::warning
Si vous supprimez ou déplacez les fichiers .zip qui sont dans ce dossier de backups, le logiciel ne les trouvera plus.
Donc, à moins d'une raison précise, essayez de gérer vos backups comme prévu à l'intérieur du logiciel.
:::

## Justificatif
À moins d'une exception ou d'un problème technique, vous ne devriez jamais avoir besoin de gérer vos copies de sauvegardes
manuellement de cette façon. Cet article est une référence au cas où vous en auriez besoin.

La bonne manière de gérer les copies de sauvegardes est expliquée dans le guide suivant:
[Copies de sauvegardes](/docs/gestion-dossiers/copies-de-sauvegarde/)
