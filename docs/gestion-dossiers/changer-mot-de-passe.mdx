La récupération d'un dossier pour lequel vous avez oublié le mot de passe est possible en quelques étapes faciles.
Par contre, vous allez avoir besoin d'une connexion à internet pour recevoir le courriel de récupération.

La récupération du mot de passe fonctionne avec un code de validation qui est envoyé à l'adresse courriel associée
au dossier de tenue de livres. (Il s'agit de l'adresse que vous avez choisie quand le dossier a été créé.)

## Voici les étapes pour modifier votre mot de passe.

### 1. Obtenir un code de validation par courriel

<Image img={forgotPassword}/>
<br/>
<br/>

<ol type="A">
    <li>Ouvrez le logiciel à la page d'accueil / connexion.</li>
    <li>Sélectionnez votre dossier de tenue de livres dans la liste déroulante.</li>
    <li>Entrez le mot de passe invalide et appuyez sur "Ouvrir le dossier".</li>
    <li>Le bouton "J'ai oublié le mot de passe" va devenir visible, cliquez sur celui-ci.</li>
</ol>

### 2. Saisir le code de validation reçu

<Image img={forgotPassword2}/>
<br/>
<br/>

<ol type="A">
    <li>Vérifiez vos messages à l'adresse courriel qui est associée au dossier Finance&nbsp;D.</li>
    <li>Copiez le code de validation reçu par courriel dans le logiciel.</li>
</ol>

### 3. Choisir un nouveau mot de passe et confirmer

<Image img={forgotPassword3}/>
<br/>
<br/>

<ol type="A">
    <li>Choisissez un nouveau mot de passe.</li>
    <li>Confirmez le nouveau mot de passe.</li>
    <li>Cliquez sur le bouton "Changer le mot de passe".</li>
</ol>


## Terminé.

Le nouveau mot de passe est maintenant actif pour votre dossier. Il est donc possible de se connecter comme d'habitude.
