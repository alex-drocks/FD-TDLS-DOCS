Voici comment changer le nom d'un dossier de tenue de livres dans Finance&nbsp;D&nbsp;:

1. [Ouvrez d'abord le dossier](/docs/bases/ouvrir-dossier) que vous voulez supprimer.
2. Ensuite, [ouvrez la fenêtre d'options](/docs/navigation/fenetre-options-preferences-parametres).
3. Cliquez sur l'onglet "Mon dossier".
4. Cliquez sur l'icône de crayon en haut à droite du nom de dossier.
5. Écrivez le nouveau nom du dossier.
6. Finalement, cliquez sur "Modifier le nom".
