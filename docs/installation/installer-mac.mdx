## 1. Cliquez sur le fichier d'installation
Après avoir téléchargé le logiciel, double-cliquez sur le fichier d'installation et attendez
de voir la fenêtre où votre Mac permettra de glisser le logiciel dans vos Applications.

## 2. Glissez Finance D dans vos Applications

Si vous avez réussi à lancer le fichier d'installation correctement,
vous devriez voir une nouvelle fenêtre avec un message similaire à
"Copie de Finance D dans vos Applications".
Avec votre souris, cliquez sur l'icône du logiciel Finance D,
et en maintenant le bouton enfoncé, glissez le vers vos "Applications".

## 3. Patientez pendant l'installation
À partir de là, votre ordinateur va installer le logiciel.
Vous aurez besoin d'attendre quelques secondes pour que le processus soit complet.

## Terminé
Lorsque l'installation est terminée, ouvrez le logiciel comme vous faites habituellement (s'il n'est pas déjà ouvert).
Excellent, vous avez réussi. Il n'y rien d'autre à faire pour l'installation.
Vous pouvez commencer à vous servir du logiciel.

Pour en savoir plus à propos du fonctionnement des bases du logiciel et pour vous assurer d'un bon départ,
consultez la section ["Les bases du logiciel"](/docs/bases/prise-en-main-du-logiciel).
