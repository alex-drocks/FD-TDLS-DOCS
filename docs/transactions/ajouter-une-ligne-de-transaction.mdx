Dans Finance D, vous pouvez ajouter autant de lignes de transactions qu'il vous est nécessaire
pour chaque facture.


## 1. Assurez-vous d'avoir une facture sans erreur
Il est possible d'ajouter autant de ligne de transactions que vous voulez.
Cependant, la facture doit d'abord être dans un état où il n'y a pas d'erreur.
Vous ne pourrez pas ajouter une ligne s'il y a une erreur dans la facture.
Si c'est le cas, corrigez l'erreur avant, et ensuite procédez à l'étape suivante.


## 2. Cliquez sur le bouton [ + ]
Pour créer une nouvelle ligne de transaction, cliquez sur le bouton qui se trouve sous la dernière ligne.

<Image img={addRow}/>
<br/>

:::info
Lorsque votre curseur est situé dans une ligne de transaction,
vous pouvez utiliser un des deux raccourcis clavier pour créer la nouvelle ligne plus rapidement.
Soit Ctrl+Enter ou la touche "+" sur votre clavier.
:::

<br/>
<br/>
<hr/>

## Terminé
Vous avez ajouté une nouvelle ligne de transaction avec succès.
Complétez-la et répétez autant de fois que nécessaire.
