Quand vous utilisez souvent des produits ou services, vous pouvez créer des modèles de transactions
pour accélérer la création de vos factures et pour vous souvenir du prix habituellement facturé
d'un bien ou d'un service.

Par exemple : Si vous vendez des pièces mécaniques à vos clients,
ce serait une excellente idée d'enregistrer celles que vous vendez
souvent comme des modèles de transaction.


## Créer un modèle de transaction, un bien, ou un service récurrent
1. <strong>Complétez une ligne de
  transaction</strong> en inscrivant le Montant, la Quantité, la Taxe, la Catégorie et la Description.
2. <strong>Enregistrez</strong> cette ligne comme un modèle avec la méthode de votre choix :

 * Cliquez sur "Créer un modèle de transaction" sous colonne de Description.
 * Faites un clic droit dans une des cellules de cette ligne puis, cliquez sur "Enregistrer modèle".
 * Cliquez sur l'icône d'options à gauche de cette ligne puis, cliquez sur "Enregistrer modèle".
 * Utilisez le raccourci clavier <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>M</kbd> quand vous êtes dans la ligne de transaction.

## Utiliser un modèle de transaction, un bien, ou un service récurrent
1. La première étape est d'<strong>ouvrir le menu qui contient vos modèles</strong>. Il existe plusieurs méthodes pour utiliser vos modèles de transactions rapidement... Choisissez celle qui vous convient le mieux parmi les suivantes :
 * Utilisez le raccourci clavier <kbd>Ctrl</kbd> + <kbd>M</kbd> quand vous êtes dans une ligne de transaction.
 * Faites un clic droit dans une des cellules d'une ligne de transaction puis, cliquez sur "Modèles de transactions".
 * Cliquez sur l'icône d'options à gauche d'une ligne puis, cliquez sur "Modèles de transactions".
 * Cliquez sur l'icône de flèche vers le bas dans la colonne de Description d'une ligne puis, cliquez sur "Modèles de transactions".
2. Une fois que le menu avec vos modèles est ouvert, <strong>double-cliquez sur le modèle de votre choix</strong>,
ou cliquez sur le bouton "Générer une transaction à partir du modèle sélectionné"

## Conseil de pro

Utilisez le raccourci clavier <kbd>Alt</kbd> + <kbd>Enter</kbd>
quand vous êtes dans une ligne de transaction,
ou <kbd>Alt</kbd> + <kbd>Clic</kbd> sur le bouton [ + ]
pour ajouter une nouvelle ligne de
transaction en ouvrant automatiquement le menu des modèles de transactions.

Voyez [l'exemple ci-dessus](/docs/transactions/modeles-de-transactions-et-produits#exemples-dutilisation-dun-modèle-de-transaction)
dans laquelle nous utilisons <kbd>Alt</kbd> + <kbd>Clic</kbd>.
