Chaque ligne de transaction a accès au menu d'options supplémentaire des transactions avec lequel
vous pouvez gagner en productivité.

Avec ce menu, il est possible de créer et utiliser des modèles de transactions pour gérer plus facilement
des items récurrents, par exemple.

Il est aussi possible de dupliquer ou supprimer des lignes.

## 1. Ouvrir le menu d'options supplémentaires
La première façon d'afficher le menu d'options d'une ligne, est de cliquer sur son numéro à gauche.
La deuxième méthode est de faire un clic droit à l'intérieur d'une cellule de cette ligne de transaction.

<Image img={menu}/>
<br/>

## 2. Cliquez sur une des options du menu
Lorsque le menu est visible, il suffit de choisir ce que vous voulez faire et ensuite cliquez sur l'option voulue.

<Image img={menuIsOpen}/>
<br/>

:::info
La fonction "Dupliquer" va créer une ligne identique en dernière position de la table des transactions.
Par exemple, si vous dupliquez la ligne #1 et que la facture contient déjà 3 lignes,
une 4e ligne identique à la #1 sera ajoutée.
:::


## TIP: Utilisez les raccourcis clavier
Les options du menu sont aussi accessibles via un raccourci clavier.
Pour les découvrir, ouvrez le menu d'options et déplacer votre souris sur les options.

Avec les raccourcis, le travail peut devenir beaucoup plus efficace.
