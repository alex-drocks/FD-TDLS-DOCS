Les catégories financières pour les revenus et les dépenses
peuvent être modifiées selon vos besoins spécifiques.
Avec le gestionnaire de catégories, vous pouvez modifier, ajouter et supprimer des catégories.

Voici comment procéder :


## Personnalisez vos catégories de revenus
1. Sélectionnez l'onglet "REVENUS" de la page principale
2. Déplacez votre souris par-dessus la colonne "CATÉGORIE FINANCIÈRE".
3. Ensuite, cliquez sur l'icône de crayon vert qui apparait pour ouvrir le gestionnaire de catégories de revenus.

## Personnalisez vos catégories de dépenses
1. Sélectionnez l'onglet "DÉPENSES" de la page principale
2. Déplacez votre souris par-dessus la colonne "CATÉGORIE FINANCIÈRE".
3. Ensuite, cliquez sur l'icône de crayon rouge qui apparait pour ouvrir le gestionnaire de catégories de dépenses.

:::note
Si une catégorie financière est utilisée dans une facture existante, cette catégorie ne peut pas être supprimée.
Elle peut seulement être modifiée.

Modifier le nom d'une catégorie mets à jour tous les enregistrements y faisant référence dans le logiciel.
C'est-à-dire, toutes factures, préférences et tous rapports seront à jour avec le nouveau nom de catégorie.

Quand vous supprimez une catégorie, un enregistrement automatique se produit.
Ce qui explique pourquoi le bouton "Enregistrer" du gestionnaire de catégorie
devient inactif après une suppression.
:::
